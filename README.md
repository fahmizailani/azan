# Rihla Chrome Extension

A base Chrome extension scaffold for fetching and displaying prayer times.

## Load locally in Chrome

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this project folder.

## Current base features

- Manifest V3 setup
- Popup UI with prayer-time fetch flow
- Background service worker with local storage
- Options page for notification setting
